# java-collections

In this repository, I am going to show various examples using Java Collections
Framework.

# Collection Framework contains
- Interface
- Implementations
- Algorithms

## Interfaces
- Collection
- Set (and SortedSet) <= collection without duplicate elements
- List <= ordered collection
- Queue and Dequeue <= collection for holding elements before processing
- Map (and SortedMap) <= maps keys to values

## Algorithms
- Sorting
- Shuffling
- Data Manipulation
- Searching
- Composition
- Extreme values

## Structure
- Iterable Interface
  - Collection Interface
--- List Interface
---- ArrayList Class
---- LinkedList Class
---- Vector Class
----- Stack Class
--- Queue Interface
---- PriorityQueue Class
---- Dequeue Interface
----- ArrayQueue Class
--- Set Interface
---- HashSet Class
---- LinkedHashSet Class
---- SortedSet Class
----- TreeSet Class
