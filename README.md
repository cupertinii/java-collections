Framework.

- Interface
- Implementations
- Algorithms

- Collection
- Set (and SortedSet)
- List
- Queue / Dequeue
- Map (and SortedMap)

# Algorithms
- Sorting
- Shuffling
- Data Manipulation
- Searching
- Composition
- Extreme values
